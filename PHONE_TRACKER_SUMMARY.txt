HOW TO USE
----------
1. Start a local HTTP server:
   python3 -m http.server 9000

2. Open in browser:
   http://localhost:9000/tracker.html

3. Enter a phone number with country code (e.g., +14155552671)

4. Click "Track Number" to see results

5. View history and click items to see past results
